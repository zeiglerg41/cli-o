# Change Log

All notable changes to the "clio-vscode" extension will be documented in this file.

## [0.1.0] - 2024-12-05

### Added
- Initial release of Clio VS Code extension
- Chat interface for interacting with Clio AI assistant
- Real-time code editing with automatic application of changes
- Diff visualization with color-coded gutter decorations
- Status bar integration showing Clio's activity
- Support for multi-file edits
- JSON-based communication protocol with clio CLI
- Webview-based chat panel with message history
- Error handling and process lifecycle management

### Features
- Start/Stop session commands
- Interactive chat with Clio
- Automatic edit application
- Visual feedback for code changes
- Process cleanup on extension deactivation

## [Unreleased]

### Planned Features
- Accept/Reject UI for proposed edits
- Inline diff preview on gutter click
- Edit history and undo functionality
- Configuration options for auto-apply behavior
- Syntax highlighting in chat messages
- File tree view for multi-file changes
- Keyboard shortcuts for common actions
